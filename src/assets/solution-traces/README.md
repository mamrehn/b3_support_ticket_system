# Referenz-Screenshots der Kollegin (optional)

Hier optionale Trace-Screenshots ablegen, die im Feedback-Kommentar der
erfahrenen Kollegin angezeigt werden – z. B. wie der Wireshark-Trace zur
korrekten Lösung aussieht.

1. Bilddatei hier ablegen, z. B. `ticket-4.png`.
2. In `src/lib/solutionTraces.ts` importieren und der Ticket-ID zuordnen.

Ohne Eintrag wird im Kommentar kein Screenshot angezeigt.
